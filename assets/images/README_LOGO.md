# Instructions pour le logo de l'établissement

## 📁 Emplacement du logo
Placez votre fichier logo dans :
```
c:\wamp64\www\Projet TATOU\assets\images\logo-etablissement.png
```

## 🎨 Spécifications recommandées
- **Format** : PNG (avec transparence) ou JPG
- **Taille** : 88x88 pixels minimum (44x44px affiché)
- **Résolution** : 300 DPI pour une meilleure qualité
- **Fond** : De préférence transparent pour s'adapter au design

## 🔄 Fallback
Si le logo ne peut pas être chargé, le système affichera automatiquement l'icône "T" par défaut.

## 📝 Étapes
1. Copiez votre logo dans le dossier `assets/images/`
2. Renommez-le en `logo-etablissement.png`
3. Actualisez la page de connexion

Le logo s'affichera à côté de "TAAJ Corp" avec une bordure dorée et sera parfaitement intégré au design.
